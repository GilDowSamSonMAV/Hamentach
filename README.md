# Multi-Agent System (Hackathon Project)

This is a multi-agent system designed for the hackathon. It utilizes a simple orchestration pattern to coordinate multiple agents.

## Structure

- `src/agents/`: Define agent logic and types here.
- `src/tools/`: Add functions/capabilities for agents to use.
- `main.py`: The entry point for orchestrating agent interactions.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the system:
   ```bash
   python main.py
   ```
